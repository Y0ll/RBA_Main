MainForLearn - Скрипт для обучения модели, данные выгружаются в NiFi из БД MySQL по расписанию раз в неделю, настройка через CRON и отправляются на обучение через http  
MainForRisk - Скрипт для оценки риска в реальном времени, модель подтягивается с файлового хранилища, пороги риска сохраняются в .txt файл  
RBA.xml - Template NiFi 1.18.0 - Шаблон разработанного ETL\ELT в NiFi
